# arvan-dashboard

## Project Setup

```sh
npm install
```

## Environment Setup

```sh
# .env in root dir
VITE_API_URL=https://api.realworld.io/api
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
